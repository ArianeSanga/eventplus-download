Baixar APK
Arquivo: apk/EventPlus.apk — faça o upload desse arquivo na pasta apk/ do repositório.

Dica: para instalar no Android, ative Instalar apps de fontes desconhecidas nas configurações do dispositivo.

  1. Faça upload do arquivo EventPlus.apk na pasta apk/ do repositório.
  2. Published GitHub Pages (Settings → Pages → branch main) → você terá uma URL: https://<seu-usuario>.github.io/<repo>/
  3. Use essa URL como fallback nos Dynamic Links do Firebase.
